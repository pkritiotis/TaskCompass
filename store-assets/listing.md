# Chrome Web Store Listing Draft

## Name

Task Compass for Google Tasks

## Short Description

Prioritize Google Tasks in a focused Eisenhower Matrix side panel.

## Detailed Description

Task Compass for Google Tasks helps you turn a busy task list into a clearer priority map.

Open Task Compass from Chrome's side panel, connect Google Tasks, choose a task list, and organize incomplete tasks by urgency and importance. Tasks due today or overdue are treated as urgent automatically, while you can drag tasks between quadrants when your context says something different.

Task Compass keeps Google Tasks as the source of truth. It reads your task lists, loads incomplete tasks, and updates Google Tasks when you mark a task complete. Your selected list and matrix choices are stored with Chrome storage so the view can follow your browser.

Features:

- View Google Tasks in an Eisenhower Matrix
- Choose which Google Tasks list to prioritize
- Drag tasks between Do, Schedule, Delegate, and Eliminate
- Mark tasks complete directly from the side panel
- Refresh the panel to fetch the latest incomplete tasks
- Disconnect Google and request OAuth token revocation
- Sync selected list and matrix metadata with Chrome storage

Task Compass does not use ads, analytics, or tracking.
